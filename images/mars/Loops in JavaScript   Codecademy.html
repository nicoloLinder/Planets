Looping Through Arrays

We've seen that loops can be used to count. We can also use them to run through all of the items in an array.

Each element in an array has a position number, or index. The first step to looping through all of the elements of an array is to loop through all of the indices. Array indices start at 0 and end at one less than the array's size (which
can be found with array.length).

If you have an index number (like 0), you can get the corresponding array item (like "cat") with:
array[indexNumber]

Instructions

Print out all of the items in the array with console.log. Not sure where to begin? Read the exercise hint.

Stuck? Get a hint! Hint

  1. Create a for loop that cycles through all of the indices of the animals array.

    a. The first element of the array is always numbered starting with 0. Thus, with the first for loop parameter, you'll want to set i to 0.

    b. The last element of the array will be one less than the length of the array. To make the loop stop there, you can use the < operator to compare i and the length of the array in the second loop parameter.

    c. The elements of an array are numbered by ones. Write the third loop parameter so that i is incremented by ones.

  2. Add a console.log call in the body of the loop that uses an array[indexNumber] statement to get the element of animals corresponding to the current value of i.
